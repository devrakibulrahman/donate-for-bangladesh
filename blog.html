Important FAQ

1. What is the Document Object Modal (DOM) ?

Document Object Model the short from is (DOM). DOM is a programming interface for web documents. DOM can change web documents style, structure, tag, element etc. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page

2. How do you select an element from the DOM ?

In JavaScript they are provide 5 different DOM access method.
This method is: getElementByClassName(), getElementByTagName(), getElementById)(), querySelector(), querySelectorAll(). Single element selector is getElementById() and querySelector(). And JavaScript also have multiple element selector is getElementByClassName(), getElementByTagName(), querySelectorAll(). I can use those method for select DOM element.

3. What is event delegation in the context of the DOM, and why is it useful ?

The process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

4. How do you manipulate an element's attributes and styles using the DOM ?

First of all select the element. Which element you like to change the style. You can select an element easy with the DOM. JavaScript provide more different type of method for styling an element. Those method is: getAttribute(), setAttribute(), removeAttribute(), style property and also have classList.add(), classList.remove(). And many more.
